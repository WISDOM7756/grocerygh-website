GROCERYGH WEBSITE

This is a simple e-commerce grocery website designed to display grocery products and allow users to browse items online.

The project was created as part of a web development task to demonstrate website layout design and product listing for an online grocery store.

## Features

- Display grocery products with images and prices
- Organized product cards
- Simple navigation interface
- Responsive layout for easy viewing

## Technologies Used

- HTML
- PHP
- CSS
- JavaScript

## Project Purpose

This project demonstrates the design of a basic online grocery store interface where customers can view available products.
